# DC Cleaner

light DC Cleaner using Node.js  
Node.js 기반 초경량 DC 클리너입니다.  

# require

[Node.js](https://nodejs.org/ko/download/)

운영체제 맞게 Node.js를 설치합니다.

# install

1. clone project  

프로젝트를 클론합니다.

```bash
$ git clone https://github.com/Einere/DC_Cleaner.git
```

2. install package

의존성 패키지를 설치합니다. 

```bash
$ npm install
```

3. make user information

```javascript
// ./user.js
const user = {
    id: 'jumper1',
    pw: 'Rmaejd@762'
};

module.exports = user;
```

# run

```bash
npm run start
```
